### 💻 كيفية تشغيل المشروع:

```bash
# 1. تثبيت المكتبات
npm install

# 2. إعداد قاعدة البيانات
cp .env.example .env
# تحديث بيانات قاعدة البيانات في ملف .env

# 3. إنشاء قاعدة البيانات
npx prisma db push
npx prisma generate

# 4. إدراج البيانات التجريبية (اختياري)
npm run db:seed

# 5. تشغيل المشروع
npm run dev
```
